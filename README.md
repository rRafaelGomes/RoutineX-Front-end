# 💜 RoutineX - Front-end

## 📖 Descrição

RoutineX é uma aplicação web desenvolvida para auxiliar usuários no gerenciamento de hábitos diários.

A interface permite cadastrar, visualizar, editar, excluir e acompanhar hábitos por meio de uma dashboard moderna e intuitiva.

O Front-end foi desenvolvido utilizando HTML, CSS e JavaScript, consumindo uma API REST desenvolvida em Flask.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

## ⚙️ Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/rRafaelGomes/RoutineX-Front-end.git
```

### 2. Acesse a pasta do projeto

```bash
cd RoutineX-Front-end
```

### 3. Execute o Front-end

Você pode abrir o arquivo **index.html** diretamente no navegador.

Ou utilizar um servidor local com Python:

```bash
python -m http.server 5500
```

Depois acesse:

```
http://localhost:5500
```

---

## 🔗 Comunicação com a API

O Front-end realiza requisições para a API através do endereço:

```
http://127.0.0.1:5000
```

Antes de utilizar a aplicação, certifique-se de que a API esteja em execução.
